# 📊 Plano de Carreira - Analista de Dados (120 dias)

Este repositório contém meu plano de estudos estruturado para me tornar **Analista de Dados**, com foco em GitHub desde o início, portfólio sólido e preparação para entrevistas.

---

## 🧩 Visão do Dia a Dia

Como é o trabalho de um(a) Analista de Dados:
- Coletar e organizar dados de diferentes fontes
- Limpar e transformar dados para análise
- Criar dashboards e relatórios para apoiar decisões
- Explorar dados para identificar padrões e insights
- Documentar processos e comunicar resultados para equipes de negócio

---

## 🧠 Mapa de Skills

**Core Skills (essenciais):**
- SQL para consultas em bancos de dados
- Estatística básica aplicada a dados
- Python para análise de dados

**Nice-to-Have (complementares):**
- Visualização de dados com Power BI ou Tableau
- Inglês técnico para leitura de documentação

**Ferramentas e Tecnologias:**
- GitHub (versionamento e portfólio)
- SQL Server / MySQL
- Python (pandas, matplotlib, scikit-learn)
- Power BI / Tableau

## 🚀 Projeto de Portfólio

**Projeto:** Dashboard de Vendas e Insights  

**O que fazer:**  
Construir pipeline que coleta dados de vendas (dataset público), organiza em SQL/Python e apresenta insights em dashboard interativo.  

**Entregáveis:**
- Script de limpeza e transformação dos dados
- Banco de dados com tabelas organizadas
- Dashboard interativo com KPIs (receita, produtos mais vendidos, regiões)
- Repositório GitHub com README detalhado

**Critérios de Aceitação:**
- Dados limpos e consistentes
- Dashboard funcional e visualmente claro
- Documentação completa no GitHub

**Dica:** Use GitHub desde o início para mostrar evolução e aprendizado contínuo.

## 💬 Roteiro de Entrevistas

**Pergunta 1:** O que faz um Analista de Dados?  
**Resposta:** Organiza, analisa e gera insights para apoiar decisões estratégicas.  

**Pergunta 2:** Qual linguagem/ferramenta você domina?  
**Resposta:** Cite SQL e Python, mostrando exemplos de projetos publicados no GitHub.  

**Pergunta 3:** Como você lida com dados faltantes ou inconsistentes?

**Pergunta 4:** Você já criou relatórios ou dashboards?  
**Resposta:** Fale sobre o projeto de portfólio e como estruturou KPIs.  

**Pergunta 5:** Qual foi seu maior aprendizado até agora?  
**Resposta:** Destaque a importância da prática, documentação técnica e uso do GitHub.
